### **02. Join two strings:**

---------------------------------------

<strong>Solution: </strong>

```python

y = "Languages"
z = x + y
print(z)
```

### **03. Format floating point in the string:**

---------------------------------------

<strong>Solution: </strong>

```python
# Use of String Formatting
x = 462.75897
print("{:5.2f}".format(x))

# Use of String Interpolation
y = 462.75897
print("%5.2f" % y)
```
